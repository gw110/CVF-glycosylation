About the file 'CVF data.csv' 

The columns dHex.count, HexNAc.count, NeuAc.count and Hex.count correspond to the number of Fuc, GlcNAc, NeuAc, and Gal that modify the glcan antennae. 

The mass column shows the M+Na of permethylated glycans

Sample.name corresponds to the Sample Code in the manuscript. For the Sample Code that started with '2V', the '2V' was removed for computer processing.
